ШАГ 1: Зайдите в панель Beget
───────────────────────────────────────────────────────────────────
  1. Откройте https://beget.com
  2. Войдите в панель управления
  3. Откройте "File Manager" (Файловый менеджер)

ШАГ 2: Найдите папку сайта planeta.marketing
───────────────────────────────────────────────────────────────────
  • Обычно это: public_html
  • Или: public_html/planeta.marketing
  • Или: domains/planeta.marketing/public_html
  
  Найдите папку где сейчас находятся файлы сайта!

ШАГ 3: УДАЛИТЕ ВСЕ СТАРЫЕ ФАЙЛЫ! (ВАЖНО!)
───────────────────────────────────────────────────────────────────
  1. Выделите ВСЕ файлы в папке (Ctrl+A или Cmd+A)
  2. Нажмите "Удалить" / "Delete"
  3. Подтвердите удаление
  
  ⚠️  Удалите ВСЁ:
     • index.html
     • assets/
     • images/
     • .htaccess (включите показ скрытых файлов!)
     • всё остальное

ШАГ 4: Загрузите НОВЫЙ архив
───────────────────────────────────────────────────────────────────
  1. Нажмите "Upload" (Загрузить)
  2. Выберите файл: planeta-beget-НОВАЯ-ВЕРСИЯ.tar.gz
  3. Дождитесь окончания загрузки (8.8 MB)

ШАГ 5: Распакуйте архив
───────────────────────────────────────────────────────────────────
  1. Кликните правой кнопкой на архив
  2. Выберите "Extract" (Извлечь) или "Распаковать"
  3. Дождитесь окончания
  4. Удалите сам архив после распаковки

ШАГ 6: Проверьте файлы
───────────────────────────────────────────────────────────────────
  Должны быть:
  ✓ index.html
  ✓ .htaccess (включите показ скрытых файлов!)
  ✓ папка assets/ (с файлами .js и .css)
  ✓ папка images/
  ✓ clear-tokens.html

ШАГ 7: ОЧИСТИТЕ КЭШ БРАУЗЕРА!
───────────────────────────────────────────────────────────────────
  • Windows/Linux: Ctrl + Shift + R
  • Mac: Cmd + Shift + R
  
  Или откройте режим инкогнито:
  • Ctrl + Shift + N (Windows/Linux)
  • Cmd + Shift + N (Mac)

ШАГ 8: Проверьте сайт
───────────────────────────────────────────────────────────────────
  Откройте: https://planeta.marketing/signin
  
  ✅ ДОЛЖНО БЫТЬ (новая версия):
     • Заголовок "Вход"
     • Подпись "Введите email и пароль для входа"
     • Сразу форма с полями Email и Пароль
     • НЕТ кнопок Google/Twitter
     • НЕТ разделителя "Или"
  
  ❌ ЕСЛИ ВСЁ ЕЩЁ СТАРАЯ ВЕРСИЯ:
     1. Проверьте что удалили ВСЕ старые файлы
     2. Проверьте что извлекли архив
     3. Очистите кэш браузера ещё раз
     4. Откройте режим инкогнито для проверки

ПРОБЛЕМА: Не могу удалить файлы
───────────────────────────────────────────────────────────────────
Решение:
  • Проверьте права доступа
  • Попробуйте через FTP (FileZilla)
  • Обратитесь в поддержку Beget

ПРОБЛЕМА: Не могу найти папку сайта
───────────────────────────────────────────────────────────────────
Решение:
  • Посмотрите в разделе "Сайты" в панели Beget
  • Там указан Document Root (корневая папка)
  • Обычно это public_html или public_html/домен

ПРОБЛЕМА: Архив не распаковывается
───────────────────────────────────────────────────────────────────
Решение:
  • Распакуйте архив на своём компьютере
  • Загрузите файлы через FTP по одному
  • См. инструкцию КАК_ОБНОВИТЬ_НА_BEGET.txt

ПРОБЛЕМА: После обновления всё равно старая версия
───────────────────────────────────────────────────────────────────
Решение:
  1. Очистите кэш: Ctrl+Shift+R
  2. Откройте режим инкогнито
  3. Подождите 5-10 минут (кэш CDN Beget)
  4. Проверьте что загрузили правильный архив
