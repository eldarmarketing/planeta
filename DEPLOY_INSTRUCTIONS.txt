🎯 СПОСОБ 1: Через File Manager (РЕКОМЕНДУЕТСЯ)

1️⃣  Зайдите на https://beget.com
    → Войдите в панель управления

2️⃣  Откройте File Manager (Файловый менеджер)

3️⃣  Перейдите в папку вашего сайта:
    обычно: public_html/ваш_домен
    или просто: public_html

4️⃣  УДАЛИТЕ все старые файлы (если есть)

5️⃣  Вариант А: Загрузить архив
    • Нажмите "Загрузить" → выберите planeta-beget.tar.gz
    • После загрузки: правый клик → "Извлечь" (Extract)
    • Удалите архив после распаковки
    
    Вариант Б: Загрузить файлы напрямую
    • Откройте папку dist/ на вашем компьютере
    • Перетащите ВСЁ содержимое в File Manager:
      ✓ index.html
      ✓ .htaccess ⚠️ ВАЖНО!
      ✓ папку assets/
      ✓ папку images/
      ✓ favicon.png

6️⃣  ✅ Готово!

🎯 СПОСОБ 2: Через FTP (FileZilla)

1️⃣  Скачайте FileZilla: https://filezilla-project.org/

    Порт: 21
    Логин: ваш логин на Beget
    Пароль: ваш пароль

3️⃣  В правой части (сервер) перейдите в:
    public_html/ваш_домен

4️⃣  В левой части откройте папку dist/

5️⃣  Перетащите ВСЁ из dist/ в правую часть

6️⃣  ✅ Готово!

🔍 ПРОВЕРКА ПОСЛЕ ЗАГРУЗКИ:

1️⃣  Откройте ваш сайт: http://ваш-домен.ru

2️⃣  Проверьте что работает:
    ✓ Главная страница открывается
    ✓ Переход на /signin работает
    ✓ Переход на /clients работает
    ✓ Прямые ссылки работают (без 404)

3️⃣  Если видите белый экран:
    • Нажмите F12 → Console
    • Посмотрите на ошибки
    • Проверьте что папка assets/ загружена

4️⃣  Если роуты не работают (404):
    • Проверьте что файл .htaccess загружен
    • Проверьте права: файлы 644, папки 755

═══════════════════════════════════════════════════════════════

🔐 НАСТРОЙКА SSL (HTTPS) - ОБЯЗАТЕЛЬНО!

1️⃣  В панели Beget → SSL

2️⃣  Выберите ваш домен

3️⃣  Нажмите "Установить Let's Encrypt" (бесплатно)

4️⃣  Подождите 5-10 минут

5️⃣  ✅ Сайт будет доступен по HTTPS!

После установки SSL сайт автоматически перенаправит на HTTPS
(это настроено в .htaccess)
